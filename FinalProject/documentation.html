Shawn Clark's igme230 Final Project Documentation

In order to complete this project, I used multiple libraries and online resources, including:

  • Anime.js
  • JQuery
  • JQueryUI
  • Font Awesome
  • W3Schools
  • StackOverflow
  • Google Fonts (Itim font)

Using these libraries in combination with flexbox, html, css, and javascript, I was able to successfully create the applications for this project. In addition to learning the Anime.js library, I had to learn how to add and remove html components through javascript and make the new shapes retain the original animations I created. I also learned how to use select and input fields, getting their values and applying them through javascript. While I did learn most of these techniques over the course of the semester, it was applying and combining them that proved to be the real challenge in the end.
