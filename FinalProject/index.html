Shawn Clark igme230 Final Project

Anime.js Canvas 01

This basic application features a canvas with basic shapes animated using Anime.js. Additionally, there are playback buttons above the canvas that allow you to play, pause, and reverse the animations; you can also drag the shapes around the canvas.

Anime.js Canvas 02

This application retains the functionality of the first application, but adds the ability to spawn in new shapes using the options in the menu.

Anime.js Canvas 03

This application retains the functionality of the first two applications, but adds the ability to adjust animation playback speed and the ability to clear the canvas of any shapes currently on it.
